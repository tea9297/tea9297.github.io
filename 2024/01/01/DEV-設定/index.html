API KEY

若要使用openAI的模型，必須輸入API KEY，可選擇openAI或是Azure openAI其一
如範例，選擇使用Azure openAI後，輸入key值與URL，按下儲存，若顯示成功，便成功添加API KEY。

若勾選左下角”Save Permanently”便會將key值儲存至config中，下次登入也可以直接使用。
